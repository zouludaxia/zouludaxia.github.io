电脑白痴的Hexo博客搭建
发表于 2020-04-24| 更新于 2020-04-27
字数总计:1k|阅读时长: 3 分钟
| 阅读量:

电脑白痴 人文关怀 作者菜但超热情 持续更新

最近发现学校里有些厉害的人都在在坚持写技术博客，平时酷爱写杂感的大侠我突然觉得，建一个个人博客，拥有一个属于自己的小树洞，存存杂感放放学习笔记是多么好的一件事情。身为大侠，自然说干就干。

大侠选择的是github的博客，原因是它定制比较简单，用markdown写博文也比较方便。虽然想通过百度搜索到自己的主页比较困难(￣_￣|||)，但刚好适合我这样安静的美男子以及自闭仔。
看了一些“纯小白”的教程，发现纯小白原来是要熟练掌握Linux系统操作命令的【手动狗头】，那么，我们非计算机专业或者计算机课程到现在还没有开始上的电脑白痴，就不配拥有自己的小窝了？
不，白痴也有人爱，刚被爱过的大侠现在就来给大家分享最最简单最详细的github+hexo博客搭建攻略。（前面搭建博客的时候都是看着知乎小哥枫叶的文章做的，真的太nice了。）

大致步骤

  1. GitHub注册与git安装
  2. git与GitHub的绑定以及文件提交法
  3. 安装node.js和Hexo框架
  4. 域名的购买和解析
  5. 主题更换以及文章发布

开始吧！

1.GitHub的注册与git安装

首先登陆GitHub官网GitHub官网，点击绿绿的那个“sign up for GitHub”，输入信息，选择免费的plan。出现问卷，不想填直接点下面的“skip this step”跳过就好了。（英文看不懂就用谷歌浏览器右键自动翻译，超方便）
最后出现以下页面，就可以开始邮箱验证了。

然后进入qq邮箱里的验证邮件，点击蓝色的Verify email address按钮,又进入了GitHub登录页面（网速卡等一会就好了）。登录后又进入验证码界面，输入验证码（已经发到你的qq邮箱里了）。随后进入创建新仓库的界面就意味着已经注册成功了。


接下来安装git，进入git官网git官网，点击download下载，选择合适的系统，开始下载。下载成功后打开安装包，点next，然后选择安装位置进入下一步，勾选第一二行，表示在桌面创建快捷方式。其余保持默认。

然后一路默认下一步，直至安装完成。

右键能出现Git Bash here即是安装成功了。在不同目录点击下右键-git bash here进入的是不同目录的配置（和Kali Linux里的终端是一个道理），之后做的时候一定要看好目录哦。

2. git与GitHub的绑定以及文件提交法

打开gitbash输入 ssh 命令，查看本机是否安装 SSH：

如此即为已安装， 然后输入 ssh-keygen -t rsa 命令，表示我们指定 RSA 算法生成密钥，然后敲四次回车键，之后就就会在C:/Users/ASUS/.ssh下生成两个文件（win10），分别为秘钥 id_rsa 和公钥 id_rsa.pub.
输入下面三条命令，可查看到公钥文件里的内容。

$ cd ~/.ssh
$ ls
$ cat id_rsa.pub

点开github右上角头像中的“Settings”，再点击 SSH and GPG keys。点击绿色的“ New SSH key”新建，将公钥右键复制粘贴到新建的ssh key中（标题空着无所谓），点击“add this key”保存。
验证是否成功，我们可以通过在 Git Bash 中输入 ssh -T git@github.com 进行检验。第一次会出现：

填“yes”，出现以下情形即为成功了。

至此我们就完成了git与GitHub的绑定。
